# BigQuery Query Viewer

## command

`BQ Query Viewer: Get SQL by Job ID`

Enter `[Location].[Job ID]`, or just `[Job ID]` if default location is provided in settings. GCP project is automatically detected from the current `gcloud` cli configuration.

## Settings (JSON)

```json
{
  "bg-query-viewer": {
    "defaultLocation": "asia-northeast1"
  }
}
```
